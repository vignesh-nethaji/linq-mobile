DefaultIfEmpty (default value)
  • Performs a specified operation to each element in a collection, while carrying the result forward.
  • This Lambda Expression sample aggregates numbers in array by multiplication.

Program
private static void Sample_DefaultIfEmpty_Lambda_DefaultValue()
{
int[ ] empty = { };
var result = empty.DefaultIfEmpty(5);
Console.WriteLine("result contains one element with a value of 5:");
Console.WriteLine(result.Count() == 1 && result.First() == 5);
}
