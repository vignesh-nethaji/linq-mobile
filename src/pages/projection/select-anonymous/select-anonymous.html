Select (anonymous)
  • Selects, projects and transforms elements in a collection.
  • This Lambda Expression sample calculates cos and sin of selected angles, and projects result into anonymously typed elements.

Program
private static void Sample_Select_Lambda_Anonymous()
{
double[] angles = new double [ ] { 30D, 60D, 90D };
var result = angles.Select(a => new { Angle = a, Cos = Math.Cos(a), Sin = Math.Sin(a) });
Console.WriteLine("Calculated value:");
foreach (var res in result)
Console.WriteLine(String.Format( "Angle {0}: Cos = {1}, Sin = {2}", res.Angle, res. Cos, res.Sin));
}
